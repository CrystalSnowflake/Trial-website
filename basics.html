What is HTML?

HTML stand for Hyper-Text Markup Language invented by Tim-Berners Lee in CERN (Counseil Europeen pour la Recherch Nuclaire). It only used .htm and .html extensions. It can be created by a simple text editor (like Notepad in Windows).

Advantages of HTML

  1. Easy to use, learn, and implment
  2. Flexible alternative to traditional presentation and tedious software
  3. Contains powerful fomatting facilities
  4. Device and platform independent
  5. Hyperlinking facility available making controlled navigation possible
  6. HTML pages can be edited easily without changing the whole document
  7. It is a kind of software which has been called 'world ware'
  8. Independent work can be done and one does not need to use any application or software vendor
  9. No expensive licence software or hardware required
  10. If compatibility with user habits and expectations and multiple platforms is the goal, then HTML is the only source delivering elegant and consistent websites

Disadvantages of HTML

  1. Does not offer any programming language features and capabilities.
  2. Easy to write 'bad' HTML containing errors.
  3. Complicated HTML code is hard to read and write. HTML code gets more complicated to make an interactive website.
  4. Easy to make mistakes.
  5. Special types of software (scripting lnaguages) are required to handle events and validations.
